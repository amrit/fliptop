# Fliptop

A super simple Ruby wrapper for the Fliptop API.

## Installation

Add this line to your application's Gemfile:

    gem 'fliptop'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fliptop

## Usage

require 'fliptop'
